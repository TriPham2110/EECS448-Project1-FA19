## Building

### Prerequisites

```
gtk3
gtkmm-3.0
pkg-config
libffi
g++
homebrew
```

### Building
#### On MacOS:
`export PKG_CONFIG_PATH=/usr/local/opt/libffi/lib/pkgconfig/:$PKG_CONFIG_PATH`
#### On Linux
If it fails, find libffi and gtkmm-3.0, then add them to your PKG_CONFIG_PATH.
#### On Windows
God help you.
